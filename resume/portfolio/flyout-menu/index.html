Flyout Menu

Interaction Notes

The demo is intentionally small: one trigger, one burst of choices, and a clear collapse after selection.

  • Notice the staggered transforms that make the menu feel physical without needing extra assets.
  • The trigger and menu items are real buttons, so the interaction has a keyboard-accessible foundation.
  • The frame keeps the older CSS experiment integrated with the rest of the portfolio.
