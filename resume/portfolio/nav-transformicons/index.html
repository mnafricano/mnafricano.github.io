Interaction Notes

This is a compact motion system for commands that share the same footprint but communicate different states.

  • Notice that the layout stays stable while the icon shape changes.
  • The JavaScript only toggles state; the visual behavior lives in CSS.
  • Pressed-state attributes keep the button state available to assistive technology.
